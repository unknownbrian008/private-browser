# Private Browser

A lightweight, privacy-focused desktop web browser built with Electron.

## Privacy Features
- No telemetry
- No persistent storage
- No permissions granted by default
- No Google services
- Uses DuckDuckGo search
- Session-only browsing

## Install
```bash
git clone https://github.com/unknownbrian008/private-browser.git
cd private-browser
npm install
npm start
